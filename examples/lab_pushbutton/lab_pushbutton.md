# lab_pushbutton

The "lab push button" project consists of using push buttons to use some methods available in the library glpi_esp8266.

# Methods used

The methods used in this project were:

- **debug**

- **NewTicketIncident**

- **FollowupTicket**

- **TaskTicket**

- **SolutionTicket**
